HINTS FOR HOME SERVICE

SINGER SEWING MACHINE -- MODEL 221

Whenever sewing difficulties are encountered, review the sections of the instruction book that pertain to the operation that you are performing, to assure you are using the machine properly. If the problem still exists, the following hints will help you to correct it:

Fault -- Machine does not sew

Symptom

Remedy

a. Needle bar does not move.

1. Make sure that motor is connected to electrical supply or treadle belt is engaged.

2. Make sure that clamp stop motion is tightened

b. Needle bar moves but machine does not stitch.

1. Replace needle. Make sure proper needle class and size is used; make sure the needle is properly inserted and check to see if machine is threaded properly.

2. Make sure that bobbin case and bobbin are properly placed in machine.

3. Make sure that bobbin case is free of lint.

c. Needle bar moves and stitches but fabric does not move.

1. Make sure feed regulator is set to desired stitch length.

2. Make sure there is sufficient presser bar pressure to hold material.

3. Remove throat plate and clean all lint from around feed dog.

d. Needle breaks.

1. Replace needle, making sure it is the proper class and size and make sure it is properly inserted. Turn handwheel over by hand several times to make sure needle does not foul throat plate, presser foot or attachment.

2. If material is being guided behind the needle, make sure that it is not being pulled too quickly.

3. Make sure that presser foot or attachment is securely fastened to presser bar.

4. Do not pull the material to one side when taking it from the machine. The needle may become bent and strike the throat plate or presser foot.

Fault -- Thread breaks

Symptom

Remedy

a. Needle thread failures.

1. Needle might be bent. Replace needle making sure it is the proper class and size, and make sure it is properly inserted.

2. Make sure the machine is properly threaded.

3. If the needle thread tension is too tight it should be made lighter.

4. Check needle hole in throat plate and presser foot to ensure that they have not been damaged.

5. Make sure bobbin and bobbin case are properly inserted in machine.

6. Make sure that threads are properly placed under presser foot when starting to sew.

b. Bobbin thread failures.

1. Make sure bobbin is wound properly.

2. Make sure bobbin case is threaded and inserted properly.

3. Check to make sure bobbin or bobbin case have not been damaged.

Fault -- Skipped Stitches

Symptom

Remedy

Failure to consistently make stitches

1. Needle might be bent. Replace needle, making sure it is the proper class and size, and make sure it is properly inserted.

2. Make sure that machine is threaded properly.

3. Make sure that material is held down firmly (particularly when doing darning or embroidery).

Fault -- Looped Stitches

Symptom

Remedy

Loose stitches occurring on underside of fabric.

1. Needle might be burred. Replace needle, ensuring that it is the proper class and size, and make sure it is properly inserted.

2. If needle thread tension is too loose, it should be tightened.

3. Make sure bobbin case area is free of lint.

4. Make sure the bobbin is not wound too full (the thread should be below the edge of the sides).

Fault -- Gathering or puckering of material

Symptom

Remedy

Material does not lie flat after sewing.

1. Needle might be bent. Replace needle, making sure it is the proper class and size, and make sure it is properly inserted.

2. If needle or bobbin thread tension is too tight it should be made lighter.

3. Make sure there is sufficient presser bar pressure to hold material.

4. Stitch length may be too long for material (particularly fine materials).

5. Make sure proper presser foot is being used.

Fault -- Incorrect Feeding

Symptom

Remedy

Material does not move properly under presser foot.

1. Presser bar pressure may be too light.

2. Presser foot or attachment may not have been properly re-assembled to presser bar.

3. Feed regulator may not be set to proper stitch length.

4. Remove throat plate and make sure lint has not accumulated on feed dog.

Fault -- Sluggish and/or noisy machine

Symptom

Remedy

Handwheel difficult to turn over manually and/or machine noise level has increased.

1. All lint should be cleaned from machine, particularly in area of bobbin case.

2. All oiling points should be oiled. Singer sewing machine oil should be used.

If the suggestions offered here do not correct your sewing problems, call your local Singer Sewing Centre or authorised Singer Dealer
