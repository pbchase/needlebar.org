THE NEEDLEBAR

Singer production figures for popular machines

By Chrys Gunther

 

This is from the data that Singer has made public for Kilbowie and Elizabethport. There are some errors in their data. It does not include data from other factories.

There is also some data missing from these figures. e.g. some of the missing serial numbers are EZ, FA, FB, FC, etc.

It  does not include production prior to 1900, which is over 16 million machines.

Total production for both factories together is 81,553,194 machines. Kilbowie 47,499,655 & Elizabethport is 34,053,539 machines.
