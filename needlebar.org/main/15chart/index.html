Singer Model 15 Comparisons

by Chrys Gunther & Barbara Lee Gates

This is a chart showing the differences between the various versions of Singer's extremely successful and long-lived Model 15 machine.NB Machines made at Singer's Scottish plant at Kilbowie use the same model numbering system except that the 'dash' is replaced with a 'K', so a Kilbowie-built Singer 15-88 would have the model number 15K88.
