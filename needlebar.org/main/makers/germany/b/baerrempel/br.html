Part Two (1918 - 1968)

Changes in Model Names:

Phoenix D / DD became Class 6

Phoenix E / EE became Class 7

Phoenix F / FF became Class 8 / 208 / 308

Phoenix M became Class 12

Phoenix MB became Class 13

Phoenix MM became Class 16

Phoenix MMF became Class 17

Teutonia A = Phoenix O = Class 11

Teutonia B = Phoenix P = Class 10

Teutonia C = Phoenix Q = Class 9

Phoenix R became Class 3 / 204 / 303

Phoenix S became Class 4
