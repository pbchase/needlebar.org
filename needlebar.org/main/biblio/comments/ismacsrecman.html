BOOK LIST RATINGS

RATERS' COMMENTS

ISMACS -- (1997) --- The Singer Recognition Manual

Privately published -- London, UK

------------------------------------------

"We received this recognition manual free with our ISMACS membership, so it definitely is good value for the money. The pictures are poor, but it was still a very handy simple reference for a new collector listing many models, giving basic dates and explanation of serial numbers. Would love to see it reprinted with better quality pictures."
